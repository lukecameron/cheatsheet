# Omarchy Hotkeys

> [Source](https://learn.omacom.io/2/the-omarchy-manual/53/hotkeys)

## Navigating

| Hotkey | Function |
| --- | --- |
| `S-Space` | Application launcher |
| `S-A-Space` | Omarchy control menu |
| `S-Escape` | Lock, suspend, relaunch, restart, shutdown computer |
| `S-W` | Close window |
| `C-A-Del` | Close all windows |
| `S-T` | Toggle window between tiling/floating |
| `S-O` | Toggle popping window into sticky'n'floating |
| `S-F` | Go full screen |
| `S-A-F` | Go full width for the window |
| `S-1/2/3/4` | Jump to specific workspace |
| `S-Tab` | Jump to next workspace |
| `S-Sh-Tab` | Jump to previous workspace |
| `S-C-Tab` | Jump to former workspace |
| `S-Sh-1/2/3/4` | Move window to workspace |
| `S-Sh-A-Arrows` | Move workspaces to directional monitor |
| `S-S` | Show scratchpad workspace overlay |
| `S-A-S` | Move window to scratchpad workspace |
| `C-1/2/3/..` | Jump to browser tab |
| `S-Arrow` | Move focus to window in direction of arrow |
| `S-Sh-Arrow` | Swap window with another in direction of arrow |
| `S-Equal` | Grow windows to the left |
| `S-Minus` | Grow windows to the right |
| `S-Sh-Equal` | Grow windows to the bottom |
| `S-Sh-Minus` | Grow windows to the top |
| `S-G` | Toggle window grouping |
| `S-A-G` | Move window out of grouping |
| `S-A-Tab` | Cycle between windows in grouping |
| `S-A-1/2/3/4` | Jump to specific window in grouping |
| `S-A-Arrow` | Move window into grouping in direction of arrow |
| `S-C-Arrow` | Move between windows inside a tiling group |
