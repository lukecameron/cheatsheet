# Omarchy Hotkeys (continued)

## Notifications

| Hotkey | Function |
| --- | --- |
| `S-,` | Dismiss latest notification |
| `S-Sh-,` | Dismiss all notifications |
| `S-C-,` | Toggle silencing notifications |
| `S-A-,` | Invoke most recent notification |

## Style

> Extra background images live in ~/.config/omarchy/current/backgrounds. All style options accessible under Style in the Omarchy menu (Super + Alt + Space).

| Hotkey | Function |
| --- | --- |
| `S-C-Sh-Space` | Pick a new theme |
| `S-C-Space` | Next background image for theme |
| `S-Backspace` | Toggle transparency on a window |

## Toggles

| Hotkey | Function |
| --- | --- |
| `S-C-I` | Toggle idle/sleep prevention |
| `S-C-N` | Toggle nightlight display temperature |
| `S-Sh-Space` | Toggle the top bar |
| `S-Mute` | Switch to next audio output |

## Ghostty Terminal

| Hotkey | Function |
| --- | --- |
| `C-Sh-E` | New split below |
| `C-Sh-O` | New split besides |
| `C-A-Arrows` | Move between splits |
| `S-C-Sh-Arrows` | Resize split by 10 lines |
| `S-C-Sh-A-Arrows` | Resize split by 100 lines |
| `C-Sh-T` | New tab |
| `C-Sh-Arrows` | Move between tabs |
| `A-Numbers` | Go to specific tab |

## File Manager

| Hotkey | Function |
| --- | --- |
| `C-L` | Go to path |
| `Space` | Preview file (arrows navigate) |
| `Backspace` | Go back one folder |
