| `S-Sh-/` | Password manager (1password) |
| `S-Sh-N` | Neovim |
| `S-Sh-C` | Calendar (HEY) |
| `S-Sh-E` | Email (HEY) |
| `S-Sh-A` | AI (ChatGPT) |

## Universal clipboard

> Usually on Linux, you need Ctrl + Shift + C/V in terminal and Ctrl + C/V elsewhere. These work everywhere.

| Hotkey | Function |
| --- | --- |
| `S-C` | Copy |
| `S-X` | Cut (not in terminal) |
| `S-V` | Paste |
| `S-C-V` | Clipboard manager |

## Capture

> For screenrecordings, hit the hotkey to start, hit it again to stop. All options accessible under Capture in the Omarchy menu (Super + Alt + Space).

| Hotkey | Function |
| --- | --- |
| `PrintScreen` | Screenshot with editing |
| `Sh-PrintScreen` | Screenshot straight to clipboard |
| `A-PrintScreen` | Screenrecord |
| `S-PrintScreen` | Color picker |
| `A-Sh-L` | Copy current URL from webapp or Chromium |
